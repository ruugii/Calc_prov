# Calculadora de gestion de ligas (esta creado para la liga de League of Legends LEC)
## Models
### TEAM
#### ATTRIBUTES
    - name
    - numeber of wins
    - number of loses
    - total of matches
    - Partidos del equipo
#### CONSTRUCTOR
##### CONSTRUCTOR 1
    - name
##### CONSTRUCTOR 2
    - name
    - wins
    - loses
#### FUNCTIONS

    - Add win
    - Add lose
##### GETTERS AND SETTERS
    - Total of matches
    - Name
    - Wins
    - Loses
    - Partidos
